Test code for some sensor
For board nucleo F446RE
Using MbedOS

To run install
make, arm-none-eabi-gcc, arm-none-eabi-newlib

To build type
make

To flash copy bin file to nucleo

If there is any problem, try erase *print* wrapper
from makefile

- AMG              --X
- CMPS             --X
- LCD              --X
- Gripper Servo    --X
- Ping             --X
